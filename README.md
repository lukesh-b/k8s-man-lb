# k8s-man-lb

**Author:** lukesh-b  
**Repository Status:** Work in Progress  

This repository demonstrates ongoing development of Kubernetes infrastructure and workload management simulating prod env.

## Repository Structure

- `architecture/` – Cluster and networking design documents
- `infrastructure/` – Namespaces, storage, and cluster setup
- `workloads/` – Reusable Kubernetes manifests for pods, deployments, services, ingress, config
- `projects/` – Real-world deployment projects
- `scripts/` – Automation scripts for setup, deployment, and scaling
- `.github/workflows/` – CI/CD workflow definitions
